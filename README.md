# James Rodriss Portfolio 🚀

A premium, state-of-the-art professional portfolio website designed to showcase web development and security expertise with a high-end digital experience.

## ✨ Key Features

- **Glassmorphic UI**: Modern sticky navigation with background blur and real-time scroll progress indicator.
- **Dynamic Animations**: 
  - Smooth "Slide & Blur" career title transitions.
  - Intersection-observed skill progress bars that animate on scroll.
  - Interactive project cards with hover-zoom effects and technology tags.
- **AI Chatbot Ready**: Specialized in integrating intelligent chatbot solutions into existing web architectures.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewports.
- **SEO Optimized**: Implements best practices for search engine visibility and semantic HTML.

## 📁 Project Structure


## 🛠️ Tech Stack

- **Core**: Semantic HTML5 & Modern JavaScript (ES6+).
- **Styling**: Vanilla CSS3 with Flexbox/Grid and custom Glassmorphism tokens.
- **Libraries**: 
  - [AOS.js](https://michalsnik.github.io/aos/) (Animate On Scroll)
  - [FontAwesome](https://fontawesome.com/) (Premium Iconography)
  - [Particles.js](https://vincentgarreau.com/particles.js/) (Interactive Background)

## 🚀 Getting Started

1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. To deploy, simply upload the root directory to your preferred static hosting service (GitHub Pages, Netlify, Vercel).
