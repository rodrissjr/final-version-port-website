## 🎨 Website Design & Aesthetics
- **Theme**: Modern "Glassmorphism" (frosted glass effects, blurs, and glows).
- **Primary Colors**: 
  - **Background**: Deep Dark Cyan/Teal (`#0a1c1f`).
  - **Accents**: Vibrant Orange (`#e34f26`) and Warm Gold (`#c3700a`).
- **Interactive Elements**:
  - **Particles Background**: A dynamic network of connected nodes that reacts to mouse movement (via Particles.js).
  - **Scroll Animations**: Elements fade, slide, and zoom into view as the user scrolls (via AOS.js).
  - **Pre-loader**: A pulsing "RODRISS" logo that appears while the site is loading.

## 📂 Site Structure & Pages
- **Landing Page (`index.html`)**: 
  - Features a large "WELCOME" call-to-action button.
  - Displays a circular profile image of James with a glowing border.
  - Quick links to social media (Facebook, Instagram, GitHub).
- **Main Portfolio (`pages/portifolio.html`)**:
  - Contains the primary navigation menu (Home, About, Services, Portfolio, Contact).
  - **Introduction Section**: A bold header with a carousel that rotates through James's professional titles (Developer, Designer, etc.).
  - **Interactive Navbar**: Stays fixed at the top with a blur effect and shows a blue scroll-progress bar.
- **CV Page (`pages/cv.html`)**: Contains professional history and downloadable resume.
- **Project Detail Pages**: Individual pages for `Project 1`, `Project 2`, and `Project 3`.

## 🛠️ Technology Stack
- **Languages**: Semantic HTML5, Vanilla JavaScript (ES6+), and Modern CSS3 (Flexbox/Grid).
- **Libraries**:
  - **AOS (Animate On Scroll)**: Handles all scroll-based entrance animations.
  - **Particles.js**: Provides the interactive "constellation" background.
  - **FontAwesome**: Used for high-quality iconography.
  - **Google Fonts**: Uses the "Poppins" font family for a clean, modern look.

## 🚀 Key Features for Users
- **Responsive Design**: The site is fully optimized for mobile, tablet, and desktop screens.
- **No-Wrap Text**: Important headings are locked to single lines to maintain the professional layout on all devices.
- **Fluid Typography**: Text automatically scales based on the screen size using `clamp()` logic.
